Développeur Application Web - Framework Symfony & Angular

L'acquisition des compétences

Ce parcours 'Développement d’application WEB : PHP, Symfony, AngularJS, Node.JS' se distingue par l'acquisition des compétences de développement orientée objet avec le framawork Symfony, mais aussi l'acquisition des compétences de développement sous AngularJS et NodeJS utilisés par les API Google et les réseaux sociaux.

Pendant la formation j'ai participé à la conception, et au développement technique de nouvelles applications WEB (Site vitrine / évènementiel / ecommerce / services / extranet / intranet / communauté / portail) adaptables à tout navigateur Web (responsive flex, Grid system, bootstrap) mais aussi sur différents terminaux : Tablette/ Smartphone/ Iphone et ipad.
